Object-Oriented Databases

Object-Oriented Databases came into favor in the early to mid 90s along with the growth in object-oriented analysis, design, and programming. Data are basically stored as objects in these types of databases, and the data are persistent (meaning that data are stored on external storage as objects to be used by applications later).

Object-Oriented Database Characteristics

  • Data can be modeled after real world objects
  • Object-Oriented design techniques such as inheritance are relatively easy to implement
  • Object Query Languages are used to query the database

Popular Object-Oriented Database Implementations

Some of the most popular object-oriented databases are:
  • Perst
  • Intersystems Cache
  • Versant Object Database
