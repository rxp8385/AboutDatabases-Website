Welcome to AboutDatabases!

This site is dedicated to providing you with a lot of the important information you need about different databases. Whether you're just getting started or a seasoned pro, you're bound to find something here of interest to you.

What is a Database?

Basically, all databases pretty much do the same things:

  • They store data that are logically related
  • They allow users to turn data (raw facts) into information (processed data that increase users' knowledge)
  • They accomplish these tasks by allowing users to perform operations on the data (see below)

What Kinds of Operations?

Database operations are known as CRUD operations. CRUD is an acronym for:

  • C - Create a database or an element in the database
  • R - Read data (meaning to access data) from the database
  • U - Update data in the database
  • D - Delete data from the database

If They All Do the Same Things, Why Are There Different Kinds of Databases?

Good question! That's really the purpose of this site--to provide info on the different types of databases and why some are better suited for specific tasks. Feel free to explore the different sections to learn more!
